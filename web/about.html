Auteur : Jérémy Ruffet (sat@airnux.fr)

Cette application web est développée bénévolement, sur mon temps libre.
Elle est sous licence libre (MIT) et accessible sur github (https://github.com/jruffet/qcmffvl).

Le contenu du questionnaire est rédigé par la commission formation de la FFVL.
Un questionnaire est considéré comme réussi si le score est d'au moins 75%.

Suivant les recommandations de la FFVL, chaque série de 10 questions est répartie comme suit : météo 30%, réglementation 20%, pilotage 20%, mécavol 20%, matériel 10%.

Pour tout problème avec une question ou ses réponses : merci de cliquer sur la question (en mode révision), puis de cliquer sur l'enveloppe.
Pour une demande d'ajout, reprenez le même mode opératoire en choisissant n'importe quelle question.

N'hésitez pas à m'envoyer des rapports de bugs, problèmes d'affichage, des suggestions ou tout simplement un petit mail sympa !
Version Web
La version web a été conçue pour être multi-support (mobile / tablette / desktop) et multi-OS (Linux / Windows / Mac...).

La totalité du questionnaire n'est chargé qu'une seule fois d'un point de vue réseau.
Cela veut dire qu'il vous suffit de charger la page sur votre mobile avant de partir, et vous pourrez vous faire d'innombrables QCMs FFVL, sans avoir jamais besoin de connexion data !

La version web a été testée comme fonctionnant correctement avec Chrome et Firefox.
Merci d'utiliser un de ces 2 navigateurs si vous expérimentez des soucis.
Application Android
L'application Android encapsule une version complètement hors ligne de cet outil de révision.
Elle ne requiert aucune permission spéciale, et n'a pas besoin d'accès à internet pour fonctionner.

L'expérience ultime pour la révision de la partie théorique des brevets de pilote : n'importe où, n'importe quand !
